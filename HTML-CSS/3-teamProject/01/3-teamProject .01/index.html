自分史上最高の自分を目指せる会社

MISSION

人生における最高点への到達をアシストし続ける

あなたにとって、『自分史上最高の自分』とはどういう状態でしょうか？

人生における最高到達点は人それぞれ。

SEアシストは社員一人一人に『自分史上最高の自分』とは何かを問い続け、一人一人”人生の最高点”への到達を実現できるようアシストし続けます。

VISION

『個の成長』が『会社の発展』に直結し、強い個を持った成長集団となる

組織全体が同じ方向を向き、切磋琢磨しながら進み続けることで、個のキャリアアップ・成長に繋がり、組織の成長・発展に直結していく。SEアシストは『強い個』が集まった成長集団を目指します。

VALUE

貢献＝報酬

報酬は、社歴や年齢、ひいては”頑張り”によって決まるものではなく、関わる人、組織、社会に”貢献する”ことによって、経験や人脈、給与などの”報酬を得る”ことができます。
